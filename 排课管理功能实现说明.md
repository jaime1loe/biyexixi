# 排课管理功能实现说明

## 功能概述

排课管理功能允许管理员安排课程时间、显示课表，并通过课程管理功能设置教师。该功能包括以下核心能力：

1. **排课列表管理** - 查看、新增、编辑、删除排课记录
2. **课表视图** - 以表格形式直观展示每周排课情况
3. **时间冲突检测** - 自动检测教室时间冲突
4. **智能筛选** - 支持按学期、星期、节次筛选

## 技术架构

### 后端API

#### 路由文件
`backend/app/routers/schedule.py`

#### 主要接口

**教室相关**
- `GET /api/classrooms` - 获取所有教室（支持筛选）
- `GET /api/classrooms/{id}` - 获取教室详情
- `GET /api/classrooms/{id}/schedule` - 获取教室课表
- `GET /api/classrooms/{id}/status` - 获取教室占用状态
- `GET /api/available-classrooms` - 查询空闲教室

**课程相关**
- `GET /api/courses` - 获取所有课程（支持筛选）
- `GET /api/courses/{id}` - 获取课程详情
- `GET /api/courses/{id}/schedule` - 获取课程课表

**排课相关**
- `GET /api/schedules` - 获取排课列表（支持筛选）
- `POST /api/schedules` - 创建排课
- `PUT /api/schedules/{id}` - 更新排课
- `DELETE /api/schedules/{id}` - 删除排课
- `GET /api/buildings` - 获取所有教学楼
- `GET /api/periods` - 获取所有节次
- `GET /api/weekdays` - 获取所有星期

### 数据模型

#### Schedule（排课表）
```python
class Schedule(Base):
    id: int                    # 排课ID
    classroom_id: int          # 教室ID（外键）
    course_id: int             # 课程ID（外键）
    day_of_week: int           # 星期：1-7（1=周一）
    period: int                # 节次：1-5
    week_start: int            # 起始周
    week_end: int              # 结束周
    semester: str              # 学期：2024-2025-1
    class_name: str           # 班级名称（可选）
    created_at: datetime       # 创建时间
    updated_at: datetime       # 更新时间
```

#### 关联关系
- `classroom` → 教室信息
- `course` → 课程信息

### 前端实现

#### 组件文件
`frontend/src/components/admin/ScheduleManagement.vue`

#### API文件
`frontend/src/api/schedule.ts`

#### 核心功能

1. **双视图模式**
   - 列表视图：详细显示每条排课记录
   - 课表视图：表格形式展示每周排课情况

2. **筛选功能**
   - 学期选择：2024-2025-1、2024-2025-2等
   - 星期选择：周一至周日
   - 节次选择：第1-5节

3. **冲突检测**
   - 创建/编辑排课时自动检查教室是否在该时段被占用
   - 支持周次范围重叠检测
   - 冲突时显示提示信息

4. **表单验证**
   - 所有必填字段验证
   - 起始周不能大于结束周

## 使用流程

### 1. 查看排课列表
1. 进入管理后台 → 课程管理 → 排课管理
2. 选择学期进行筛选
3. 可选：按星期、节次进一步筛选
4. 切换"列表视图"或"课表视图"查看

### 2. 新增排课
1. 点击"新增排课"按钮
2. 填写排课信息：
   - 选择学期
   - 选择课程（显示课程代码和名称）
   - 选择教室（显示容量信息）
   - 选择星期和节次
   - 设置周次范围
   - 输入班级名称（可选）
3. 系统自动检测时间冲突
4. 无冲突时确认保存

### 3. 编辑排课
1. 在列表中找到要编辑的排课记录
2. 点击"编辑"按钮
3. 修改需要调整的信息
4. 保存时再次检测冲突

### 4. 删除排课
1. 在列表中找到要删除的排课记录
2. 点击"删除"按钮
3. 确认删除操作

### 5. 设置课程教师
1. 进入管理后台 → 课程管理 → 课程列表
2. 创建或编辑课程时选择授课教师
3. 教师信息会显示在排课记录中

## 冲突检测逻辑

系统会检查以下冲突：
1. **教室冲突**：同一教室在同一时段不能有多门课程
2. **周次重叠**：如果新排课与已有排课的周次范围有重叠，则视为冲突

检测条件：
```python
week_start <= existing.week_end AND week_end >= existing.week_start
```
