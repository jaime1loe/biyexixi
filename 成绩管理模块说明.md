# 成绩管理模块说明

## 功能概述

成绩管理模块为校园服务提供了完整的学生成绩查询和教师成绩上传功能。

### 学生功能
1. **成绩查询**：学生可以按学期查询自己的所有课程成绩
2. **GPA计算**：自动计算学期平均绩点
3. **成绩详情**：显示课程代码、名称、学分、成绩、绩点、等级等信息
4. **补考标识**：显示成绩是否为补考成绩

### 教师功能
1. **课程列表**：教师可以查看自己授课的所有课程
2. **学生名单**：查看选课学生名单和已有成绩
3. **成绩录入**：批量录入学生成绩
4. **成绩修改**：修改已录入的成绩
5. **成绩删除**：删除错误的成绩记录

## 后端API

### 学生端API

#### 查询我的成绩
```
GET /api/grades/student/my?semester=2024-2025-2
```

**响应示例：**
```json
[
  {
    "semester": "2024-2025-2",
    "course_code": "CS101",
    "course_name": "计算机导论",
    "credits": 3.0,
    "score": 85.5,
    "grade_point": 3.7,
    "grade_level": "A",
    "is_makeup": false,
    "gpa": 3.45
  }
]
```

### 教师端API

#### 查询授课课程
```
GET /api/grades/teacher/courses?semester=2024-2025-2
```

**响应示例：**
```json
[
  {
    "id": 1,
    "course_code": "CS101",
    "course_name": "计算机导论",
    "semester": "2024-2025-2",
    "credits": 3.0,
    "student_count": 45
  }
]
```

#### 获取课程学生列表
```
GET /api/grades/teacher/course/{course_id}/students?semester=2024-2025-2
```

**响应示例：**
```json
[
  {
    "student_id": 3,
    "student_name": "张三",
    "student_number": "2021001",
    "score": 85.5,
    "grade_level": "A",
    "is_makeup": false,
    "grade_id": 123
  }
]
```

#### 上传学生成绩
```
POST /api/grades/teacher/upload
```

**请求体：**
```json
{
  "student_ids": [3, 4, 5],
  "course_id": 1,
  "semester": "2024-2025-2",
  "scores": [85.5, 92.0, 78.5]
}
```

#### 修改学生成绩
```
PUT /api/grades/teacher/grade/{grade_id}
```

**请求体：**
```json
{
  "score": 90.0,
  "is_makeup": false
}
```

#### 删除学生成绩
```
DELETE /api/grades/teacher/grade/{grade_id}
```

## 数据模型

### Grade模型
```python
class Grade(Base):
    id: int  # 成绩ID
    student_id: int  # 学生ID
    course_id: int  # 课程ID
    semester: str  # 学期
    score: float  # 成绩
    grade_point: float  # 绩点
    grade_level: str  # 等级 A/B/C/D/F
    is_makeup: bool  # 是否补考
    makeup_score: float  # 补考成绩
    teacher_id: int  # 录入教师ID
    created_at: datetime
    updated_at: datetime
```

### 绩点计算规则
- 90-100分: 4.0 (A)
- 85-89分: 3.7 (A-)
- 82-84分: 3.3 (B+)
- 78-81分: 3.0 (B)
- 75-77分: 2.7 (B-)
- 72-74分: 2.3 (C+)
- 68-71分: 2.0 (C)
- 66-67分: 1.7 (C-)
- 64-65分: 1.3 (D+)
- 60-63分: 1.0 (D)
- 60分以下: 0.0 (F)

## 前端使用

### 学生成绩查询页面
1. 进入"校园服务" -> "成绩查询"
2. 选择学期（可选）
3. 点击"查询"按钮
4. 查看成绩列表和平均绩点

### 教师成绩录入页面
1. 进入"校园服务" -> "成绩查询"
2. 选择学期
3. 点击"查询课程"按钮
4. 选择要录入成绩的课程
5. 在弹出的对话框中为学生录入成绩
6. 点击"提交成绩"按钮保存

## 数据初始化

### 运行模拟数据生成脚本
```bash
cd backend
python create_mock_grades.py
```

该脚本会：
- 为每个学生随机选择3-5门课程
- 生成60-100分的随机成绩
- 自动计算绩点和等级
- 设置随机教师ID

### 测试API
```bash
cd backend
python test_grades_api.py
```

## 权限控制

- 学生只能查询自己的成绩
- 教师只能查询和修改自己授课课程的成绩
- 管理员可以查看所有成绩

## 注意事项

1. 需要确保courses表有teacher_id字段
2. 如果courses表缺少teacher_id字段，请运行：
   ```bash
   python add_teacher_id_to_courses.py
   ```
3. 学生和教师用户需要先登录才能使用相关功能
4. 成绩录入后会自动计算绩点和等级
5. 相同学生、课程、学期的重复成绩会被更新
