# 图书馆功能说明

## 功能概述
系统已集成图书馆图书查询功能，学生可以通过校园服务模块查询图书馆藏书。

## 数据库设计

### Book 表结构
```python
class Book(Base):
    """图书表"""
    __tablename__ = "books"

    id = Column(Integer, primary_key=True, index=True)
    title = Column(String(200), nullable=False)              # 书名
    author = Column(String(100))                            # 作者
    publisher = Column(String(100))                         # 出版社
    publish_year = Column(Integer)                           # 出版年份
    isbn = Column(String(20))                              # ISBN
    category = Column(String(50))                           # 分类
    location = Column(String(50))                           # 馆藏位置
    total_copies = Column(Integer, default=1)               # 总复本数
    available_copies = Column(Integer, default=1)           # 可借复本数
    status = Column(String(20), default="可借")             # 状态：可借/借出/预约/维护
    language = Column(String(20))                          # 语言：中文/英文等
    pages = Column(Integer)                                # 页数
    price = Column(Float)                                  # 价格
    description = Column(Text)                             # 简介
    created_at = Column(DateTime, default=datetime.now)
    updated_at = Column(DateTime, default=datetime.now, onupdate=datetime.now)
```

## API 接口

### 1. 查询图书列表
**接口**: `GET /api/campus/library/books`

**参数**:
- `keyword` (可选): 搜索关键词（书名或作者）
- `category` (可选): 图书分类

**响应**:

## 前端功能

### 图书馆查询界面
学生可以通过以下方式查询图书：
1. **关键词搜索**：输入书名或作者进行模糊搜索
2. **分类筛选**：按图书分类进行筛选
3. **查看详情**：点击图书查看详细信息

### 图书信息展示
- 书名、作者、出版社
- 出版年份、ISBN
- 图书分类
- 馆藏位置
- 复本数（总数/可借）
- 借阅状态

## 使用说明

### 学生使用流程
1. 进入"校园服务"页面
2. 点击"图书馆"卡片
3. 在"书籍检索"标签页
4. 输入关键词或选择分类
5. 点击"搜索"按钮
6. 查看搜索结果

### 座位查询
在"座位查询"标签页可以查看图书馆的座位情况：
- 剩余座位：156
- 总座位：500
