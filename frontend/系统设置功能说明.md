# 系统设置功能说明文档

## 功能概述

系统设置页面提供了完整的用户个性化配置功能，包括外观、语言、通知、隐私和高级设置。所有设置都会自动保存到本地存储，刷新页面后依然生效。

## 功能列表

### 1. 外观设置

#### 主题模式
- **浅色主题**: 适合日间使用的明亮界面
- **深色主题**: 适合夜间使用的暗色界面，减少眼部疲劳
- **跟随系统**: 根据操作系统的主题设置自动切换

**使用方法**:
```javascript
import { applyTheme } from '@/views/Settings'
applyTheme('dark') // 切换到深色主题
applyTheme('light') // 切换到浅色主题
applyTheme('auto') // 跟随系统主题
```

#### 主题颜色
- 支持自定义主题颜色
- 提供10种预定义颜色可选
- 实时预览效果

**使用方法**:
```javascript
document.documentElement.style.setProperty('--el-color-primary', '#409EFF')
```

#### 字体大小
- 支持 12px - 20px 范围调节
- 实时预览效果
- 全局生效

**使用方法**:
```javascript
document.documentElement.style.setProperty('--el-font-size-base', '16px')
```

#### 界面缩放
- 支持 75% - 125% 缩放比例
- 适用于不同分辨率屏幕

### 2. 语言设置

#### 界面语言
- 简体中文
- English

#### AI回复语言
- 自动检测
- 简体中文
- English

### 3. 通知设置

#### 桌面通知
- 需要浏览器授权通知权限
- 显示新回复、系统公告等通知
- 支持免打扰模式

**使用方法**:
```typescript
import { notification } from '@/utils/notification'

// 显示通知
notification.show({
  title: '新消息',
  body: '您有新的回复',
  type: 'answer',
  icon: '/favicon.ico',
  onClick: () => {
    // 点击通知后的回调
    console.log('通知被点击')
  }
})

// 设置通知开关
notification.setEnabled(true)

// 获取通知状态
const status = notification.getStatus()
```

#### 声音提醒
- 使用 Web Audio API 生成提示音
- 支持多种提示音类型

**使用方法**:
```typescript
import { sound } from '@/utils/sound'

// 播放通知音
sound.playNotificationSound()

// 播放成功音
sound.playSuccessSound()

// 播放错误音
sound.playErrorSound()

// 设置声音开关
sound.setEnabled(true)
```

#### 通知类型
- 收到新回复
- 推荐内容
- 系统公告
- 功能更新

#### 免打扰时段
- 设置开始和结束时间
- 在指定时间段内不显示通知
- 支持跨天设置（如 22:00 - 08:00）

### 4. 隐私设置

#### 公开资料
- 公开: 其他用户可以查看你的基本信息
- 私密: 仅自己可见

#### 显示在线状态
- 显示: 其他用户可以看到你在线
- 隐藏: 其他用户看不到你的在线状态

#### 历史记录可见性
- 仅自己可见
- 好友可见
- 公开

#### 数据共享
- 允许: 匿名数据用于改进服务
- 不允许: 不共享任何数据

### 5. 高级设置

#### 自动回复
- 开启后自动回复消息

#### 快捷回复短语
- 预设常用回复短语
- 支持添加和删除快捷回复
- 点击标签即可发送快捷回复

#### 清空缓存
- 清空 localStorage（保留用户信息和设置）
- 清空 sessionStorage
- 清空 IndexedDB 数据库
- 操作不可逆

#### 导出数据
- 导出问答历史和设置
- JSON 格式
- 包含时间戳

#### 删除账户
- 需要输入 "DELETE" 确认
- 删除后无法恢复
- 自动跳转到登录页

## 工具函数

### 声音工具 (`@/utils/sound`)

```typescript
import { sound, playNotificationSound, playSuccessSound, playErrorSound } from '@/utils/sound'

// 方法1: 使用单例
sound.playNotificationSound()
sound.setEnabled(true)

// 方法2: 使用便捷函数
playNotificationSound()
playSuccessSound()
playErrorSound()
```

**API**:
- `playNotificationSound()`: 播放通知音
- `playSuccessSound()`: 播放成功提示音
- `playErrorSound()`: 播放错误提示音
- `setEnabled(enabled)`: 设置声音开关
- `isEnabled()`: 获取声音状态

### 通知工具 (`@/utils/notification`)

```typescript
import {
  notification,
  showNotification,
  setNotificationEnabled,
  setNotificationSoundEnabled,
  setNotificationDndMode,
  setNotificationTypes,
  getNotificationStatus
} from '@/utils/notification'

// 方法1: 使用单例
notification.show({
  title: '标题',
  body: '内容',
  type: 'answer'
})

// 方法2: 使用便捷函数
showNotification({
  title: '标题',
  body: '内容',
  type: 'answer'
})

// 设置通知
setNotificationEnabled(true)
setNotificationSoundEnabled(true)
setNotificationDndMode(true, '22:00', '08:00')
setNotificationTypes(['answer', 'system'])

// 获取状态
const status = getNotificationStatus()
```

**API**:
- `show(options)`: 显示通知
  - `title`: 通知标题
  - `body`: 通知内容
  - `type`: 通知类型 ('answer' | 'recommend' | 'system' | 'update')
  - `icon`: 通知图标
  - `onClick`: 点击回调
- `setEnabled(enabled)`: 设置通知开关
- `setSoundEnabled(enabled)`: 设置声音开关
- `setDndMode(enabled, start, end)`: 设置免打扰模式
- `setNotificationTypes(types)`: 设置通知类型
- `getStatus()`: 获取通知状态
- `setStatus(status)`: 批量设置通知状态

## 在其他组件中使用

### 示例1: 在消息组件中使用通知

```typescript
import { notification, sound } from '@/utils/notification'
import { sound as soundManager } from '@/utils/sound'

// 收到新消息时
function onNewMessage(message: any) {
  // 播放声音
  if (soundManager.isEnabled()) {
    sound.playNotificationSound()
  }

  // 显示通知
  notification.show({
    title: '新消息',
    body: `${message.sender}: ${message.content}`,
    type: 'answer',
    onClick: () => {
      // 跳转到消息详情
      router.push(`/messages/${message.id}`)
    }
  })
}
```

### 示例2: 在表单提交后使用提示音

```typescript
import { sound } from '@/utils/sound'

async function submitForm() {
  try {
    await api.submit(data)
    sound.playSuccessSound()
    ElMessage.success('提交成功')
  } catch (error) {
    sound.playErrorSound()
    ElMessage.error('提交失败')
  }
}
```

### 示例3: 在问答组件中使用快捷回复

```typescript
import { ref } from 'vue'
import { sound } from '@/utils/sound'

const quickReplies = ref(['好的', '收到', '谢谢', '需要帮助吗?'])

function sendQuickReply(reply: string) {
  // 发送回复
  chatApi.send(reply)

  // 播放声音
  sound.playNotificationSound()

  ElMessage.success('回复已发送')
}
```

## 数据存储

所有设置都保存在 localStorage 中，key 为 `settings`：

```typescript
interface Settings {
  appearance: {
    theme: 'light' | 'dark' | 'auto'
    primaryColor: string
    fontSize: number
    scale: number
  }
  language: {
    current: 'zh-CN' | 'en-US'
    aiLanguage: 'auto' | 'zh-CN' | 'en-US'
  }
  notification: {
    desktop: boolean
    sound: boolean
    types: string[]
    dndMode: boolean
    dndStart: string
    dndEnd: string
  }
  privacy: {
    publicProfile: boolean
    showOnline: boolean
    historyVisibility: 'private' | 'friends' | 'public'
    dataSharing: boolean
  }
  advanced: {
    autoReply: boolean
    quickReplies: string[]
  }
}
```

## 注意事项

1. **浏览器兼容性**
   - 桌面通知功能需要 HTTPS 或 localhost 环境
   - Web Audio API 在部分旧浏览器中可能不支持
   - 建议使用现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）

2. **权限管理**
   - 首次使用桌面通知需要用户授权
   - 如果用户拒绝授权，无法再次请求
   - 用户可以在浏览器设置中重置通知权限

3. **性能优化**
   - 声音播放使用 Web Audio API，无需加载外部音频文件
   - 通知会在5秒后自动关闭
   - 大量通知可能影响性能，建议限制频率

4. **隐私保护**
   - 所有设置保存在本地，不会上传到服务器
   - 清空缓存会保留用户信息和设置
   - 删除账户操作需要二次确认
