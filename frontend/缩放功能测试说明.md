# 界面缩放功能说明

## 实现原理

界面缩放功能通过以下两种方式实现：

### 1. CSS 变量 + rem 单位（主要方式）

```css
:root {
  --scale-factor: 1;
}

html {
  font-size: calc(16px * var(--scale-factor));
}

body {
  font-size: calc(14px * var(--scale-factor));
}
```

当用户在设置中修改缩放比例时，JavaScript会更新 `--scale-factor` 变量：

```javascript
document.documentElement.style.setProperty('--scale-factor', scale.toString())
```

所有使用 `rem` 单位的元素都会自动根据这个变量进行缩放。

### 2. 根元素字体大小调整（辅助方式）

```javascript
document.documentElement.style.fontSize = `${scale * 16}px`
```

这会改变浏览器的基础字体大小，影响所有使用 `rem` 单位的元素。

## 使用方法

### 在系统设置中调整缩放

1. 打开 **系统设置** 页面
2. 选择 **外观设置** 标签
3. 找到 **界面缩放** 选项
4. 选择缩放比例：
   - 75% - 适合高分辨率屏幕
   - 90% - 稍微缩小界面
   - 100% - 正常大小（默认）
   - 110% - 稍微放大界面
   - 125% - 适合低分辨率屏幕

### 在代码中使用缩放

```javascript
// 导入设置
import { ref } from 'vue'

// 获取当前缩放比例
const appearance = ref({
  scale: 1.0 // 100%
})

// 手动设置缩放
document.documentElement.style.setProperty('--scale-factor', '1.25')
```

## 注意事项

### 1. 响应式设计

使用 `%`、`vw`、`vh` 等单位的元素不受缩放影响。建议在需要缩放的元素上使用 `rem` 单位：

```css
/* ✅ 推荐使用 rem */
.container {
  width: 30rem;
  padding: 1rem;
  font-size: 1rem;
}

/* ❌ 不推荐使用 px（不会缩放） */
.container {
  width: 480px;
  padding: 16px;
  font-size: 16px;
}
```

### 2. 浏览器兼容性

- ✅ Chrome 31+
- ✅ Firefox 34+
- ✅ Safari 7+
- ✅ Edge 12+
- ✅ Opera 20+

### 3. 已知的限制

1. **固定像素的元素不会缩放**
   - 如果元素使用固定 `px` 值，不会受到缩放影响
   - 解决方法：将 `px` 改为 `rem`

2. **图片大小**
   - 图片默认不会随缩放变化
   - 可以通过 CSS 设置图片为 `rem` 单位：

   ```css
   img {
     width: 20rem;
     height: auto;
   }
   ```

3. **第三方组件**
   - 部分第三方组件使用固定像素，可能无法完美缩放
   - Element Plus 组件支持 `rem` 单位，可以正常缩放

### 4. 最佳实践

```css
/* 基础样式使用 rem */
body {
  font-size: 1rem;
  line-height: 1.5;
  padding: 1rem;
}

/* 容器使用 rem */
.container {
  max-width: 60rem;
  margin: 0 auto;
  padding: 2rem;
}

/* 组件使用 rem */
.card {
  padding: 1.5rem;
  border-radius: 0.5rem;
  margin-bottom: 1rem;
}

/* 间距使用 rem */
.section {
  margin-bottom: 2rem;
}

/* 边框可以使用 rem */
.button {
  border: 0.0625rem solid #ccc; /* 1px = 0.0625rem */
}
```

## 故障排除

### 问题：缩放后界面布局错乱

**原因**：部分元素使用固定像素

**解决方法**：
1. 检查 CSS 中是否有使用 `px` 的地方
2. 将 `px` 转换为 `rem`（16px = 1rem）
3. 或使用相对单位 `%`

### 问题：缩放后某些组件大小不变

**原因**：第三方组件库使用固定像素

**解决方法**：
1. 检查组件库文档
2. 使用组件库提供的样式覆盖功能
3. 或使用 transform scale 作为备用方案

### 问题：缩放后字体模糊

**原因**：浏览器渲染问题

**解决方法**：
1. 使用整数缩放比例（75%、100%、125%）
2. 确保浏览器缩放为 100%
3. 清除浏览器缓存

## 技术细节

### rem 单位

`rem`（root em）是相对于根元素（`html`）字体大小的单位：

- `1rem` = 根元素的 `font-size`
- 默认情况下，`1rem` = `16px`
- 如果根元素 `font-size` = `20px`，则 `1rem` = `20px`

### calc() 函数

`calc()` 函数允许在 CSS 中进行数学计算：

```css
font-size: calc(16px * var(--scale-factor));
```

当 `--scale-factor` = `1.25` 时：
```css
font-size: calc(16px * 1.25); /* = 20px */
```

### CSS 变量

CSS 变量（自定义属性）允许在 CSS 中存储值：

```css
:root {
  --scale-factor: 1;
}
```

通过 JavaScript 修改：
```javascript
document.documentElement.style.setProperty('--scale-factor', '1.25')
```
