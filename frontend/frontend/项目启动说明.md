# 项目启动说明

## 项目简介
本项目是一个基于大模型智能体的高校知识库在线答疑系统，包含前端和后端两部分。

## 技术栈
- **前端**: Vue 3 + TypeScript + Element Plus + Vite + Pinia
- **后端**: FastAPI + Python + MySQL + SQLAlchemy
- **数据库**: MySQL 8.0+

## 环境要求
- Python 3.8+
- Node.js 16+
- MySQL 8.0+

- 端口: 3306
- 数据库名: qa_system

## API接口列表

### 认证接口 `/api/auth`
- `POST /register` - 用户注册
- `POST /login` - 用户登录
- `GET /me` - 获取当前用户信息
- `PUT /me` - 更新用户信息
- `GET /users` - 获取用户列表（仅管理员）

### 问答接口 `/api/questions`
- `POST /` - 提问
- `GET /my` - 获取我的问题列表
- `GET /` - 获取问题列表
- `GET /{question_id}` - 获取问题详情

### 知识库接口 `/api/knowledge`
- `POST /` - 添加知识（仅管理员）
- `POST /upload` - 上传文件并添加知识（仅管理员）
- `PUT /{knowledge_id}` - 更新知识（仅管理员）
- `DELETE /{knowledge_id}` - 删除知识（仅管理员）
- `GET /` - 获取知识列表
- `GET /{knowledge_id}` - 获取知识详情
- `GET /search` - 搜索知识
- `GET /categories` - 获取所有分类

### 统计接口 `/api/statistics`
- `GET /overview` - 获取数据概览
- `GET /daily` - 获取每日统计
- `GET /category` - 获取分类统计
- `GET /top-questions` - 获取热门问题

### 反馈接口 `/api/feedback`
- `POST /` - 提交反馈

## 常见问题

### 1. 数据库连接失败
- 检查MySQL服务是否启动
- 确认数据库用户名和密码是否正确
- 确认数据库 `qa_system` 是否已创建

### 2. 后端启动失败
- 确认Python版本是否为3.8+
- 运行 `pip install -r requirements.txt` 安装依赖
- 检查端口8000是否被占用

### 3. 前端启动失败
- 确认Node.js版本是否为16+
- 运行 `npm install` 安装依赖
- 检查端口5173是否被占用

### 4. API请求失败
- 确认后端服务已启动
- 检查 `vite.config.ts` 中的代理配置是否正确
- 打开浏览器控制台查看错误信息
