# 前端项目详细启动步骤

## 第一步: 安装 Node.js

### 1. 下载安装 Node.js
- 访问官网: https://nodejs.org/
- 下载 LTS 版本(长期支持版)
- 运行安装程序,一路点击"下一步"完成安装

### 2. 验证安装
打开命令提示符(CMD)或 PowerShell,执行以下命令:
```bash
node -v
npm -v
```

看到版本号说明安装成功,例如:
```
node -v
v18.17.0

npm -v
9.6.7
```

---

## 第二步: 进入前端项目目录

### 1. 打开命令提示符
- 按 `Win + R` 键
- 输入 `cmd` 并回车

### 2. 进入项目目录
```bash
# 进入桌面
cd Desktop

# 进入项目文件夹
cd 2026毕业实习\mypro

# 进入前端目录
cd frontend
```

或者直接输入完整路径:
```bash
cd c:\Users\19719\Desktop\2026毕业实习\mypro\frontend
```

---

## 第三步: 安装项目依赖

### 1. 使用国内镜像加速(推荐,速度快)
```bash
npm install --registry=https://registry.npmmirror.com
```

### 2. 如果使用默认源(速度可能较慢)
```bash
npm install
```

### 3. 等待安装完成
安装过程需要几分钟,会看到大量下载日志:
```
added 852 packages in 2m 30s
```

### 4. 如果安装失败
清除缓存后重试:
```bash
npm cache clean --force
npm install --registry=https://registry.npmmirror.com
```

---

## 第四步: 启动开发服务器

### 1. 启动命令
```bash
npm run dev
```

### 2. 看到以下信息说明启动成功:
```
  VITE v5.0.0  ready in 234 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### 3. 不要关闭这个命令窗口!
保持窗口打开,开发服务器会持续运行。

## 常见问题解决

### 问题1: 'npm' 不是内部或外部命令
**原因**: Node.js 未安装或未配置环境变量

**解决方法**:
1. 重新安装 Node.js
2. 安装后重启电脑
3. 再次执行 `npm -v` 验证

---

### 问题2: 端口 5173 被占用
**错误信息**: `Address already in use`

**解决方法1: 修改端口**
编辑 `vite.config.ts` 文件:
```typescript
server: {
  port: 5174,  // 改成其他端口
  // ...
}
```

**解决方法2: 关闭占用进程**
```bash
# 查找占用进程
netstat -ano | findstr :5173

# 结束进程(使用查到的PID)
taskkill /F /PID <进程ID>
```

---

### 问题3: npm install 安装很慢
**解决方法**: 使用淘宝镜像
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

---

### 问题4: 浏览器显示空白页面
**原因**: 依赖未完全安装

**解决方法**:
```bash
# 停止服务器 (Ctrl + C)
# 重新安装依赖
rm -rf node_modules
npm install --registry=https://registry.npmmirror.com
# 重新启动
npm run dev
```

---

### 问题5: API 请求失败(404 错误)
**原因**: 后端服务未启动

**解决方法**:
1. 等待后端开发完成
2. 启动后端服务(默认端口 8080)
3. 确认后端服务正常运行

---

### 问题6: 控制台报错 "Cannot find module"
**原因**: 某个依赖包未安装

**解决方法**:
```bash
npm install <缺失的包名>
# 例如
npm install axios
```

### 1. 在运行 `npm run dev` 的命令窗口
### 2. 按 `Ctrl + C` 键
### 3. 输入 `y` 确认退出

## 重新启动

如果修改了配置文件(如 vite.config.ts),需要重启:

```bash
# 1. 停止服务器 (Ctrl + C)
# 2. 重新启动
npm run dev
```
