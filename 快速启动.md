# 快速启动指南

## 一键启动（推荐）

双击运行 `启动系统.bat` 即可！

## 手动启动

### 1. 启动后端

```bash
cd backend
python main.py
```

### 2. 启动前端（新窗口）

```bash
cd frontend
npm run dev
```

## 访问地址

- 前端界面：http://localhost:5173
- 后端API：http://localhost:8000
- API文档：http://localhost:8000/docs
